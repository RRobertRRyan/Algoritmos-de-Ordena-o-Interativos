Interpretando os resultados
Comparações medem o trabalho de decisão do algoritmo. Trocas/Movimentos medem a reorganização. Merge Sort e Quick Sort têm O(n log n) — crescem muito mais lentamente. Teste com 50 elementos para ver a diferença claramente.
Todos os algoritmos são calculados pelo backend Python.

Por que algoritmos de ordenação importam?

⚡
Performance Real
A diferença entre O(n²) e O(n log n) é enorme. Para 1 milhão de elementos, Bubble Sort faz 10¹² operações enquanto Merge Sort faz apenas ~20 milhões.
🐍
Backend Python
Todos os 6 algoritmos são implementados em Python puro com geradores (yield). O Flask calcula os passos e envia ao frontend via API REST.
🌍
Dados do Mundo Real
Dados reais do Banco Mundial (ONU) — PIB, expectativa de vida, população e PIB per capita — tornam a aprendizagem concreta e significativa.
